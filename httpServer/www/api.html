API

Current list of sample rest api:

POST

/login/

							
{
	"user":"%s",
	"password":"%s"
}
							
							
Response:

							
{
	"key":"%s",
	"result":"%s"
}

GET

/config/

							
{
	"user_id":"%s",
	"token":"%s"
}

{
	"config": {
		"color": "%s",
		"background": "%s",
		"ring": "%d",

	}
}
